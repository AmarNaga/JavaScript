This page includes JavaScript code that adds and removes your favourite places to the list.

To view the list
let array=["Biratnagar", "Lalitpur", "Patan", "Kathmandu","Pokhara","Palpa"];
console.log(array);


To add items to the list
array.push("Dang", "Palpa");
console.log(array);


To remove Kathmandu from list and display the length of array.
array.splice(3,1);
console.log(array.length)
